NEIS API → 노션 DB 자동화 도구 | 최종 업데이트: 2026-02-26

메뉴 구성

메뉴기능
1. 날짜 페이지 생성해당 월 평일(월~금) 페이지를 노션 DB에 생성. 화목→시간표 "1", 월수금→시간표 "2" 자동 입력
2. 급식 메뉴 업데이트NEIS API에서 급식 데이터를 가져와 기존 노션 페이지에 메뉴 업데이트
3. 시간표 이미지 삽입각 페이지의 시간표 속성값(1~8)을 읽어 해당 이미지를 페이지에 삽입/교체
전체 실행 (1→2→3)위 3단계를 순서대로 한 번에 실행

기본 워크플로우

A. 매월 새 달 준비 (전체 실행)
1. [급식관리] > [전체 실행 (1→2→3)] 클릭
2. 연월 입력 (예: 202603)
3. 자동으로 날짜 생성 → 급식 입력 → 시간표 이미지 삽입
B. 시간표 수정이 필요한 경우
1. 노션 DB에서 해당 날짜의 시간표 속성값을 변경 (예: 2 → 5)
2. [급식관리] > [3. 시간표 이미지 삽입] 실행
3. 변경된 값에 해당하는 이미지로 자동 교체됨 (같은 이미지면 스킵)

자동 트리거

scheduledNextMonthUpdate 함수가 트리거에 등록되어 있습니다.
매월 20일~말일 사이에 매일 자동 실행되며, 다음 달 날짜 생성 + 급식 업데이트를 수행합니다.
20일 이전에는 자동 스킵됩니다.

주의 사항

노션 DB 필수 속성: 이름(제목), 날짜(날짜), 메뉴(텍스트), Month(선택), 시간표(선택, 값: 1~8)

급식 데이터 없는 달: NEIS에 데이터가 아직 등록되지 않은 경우 2단계가 0건으로 나옵니다. 추후 재실행하면 됩니다.

권한 오류: 스크립트를 처음 실행할 때 구글 권한 승인 팝업에서 「허용」을 눌러야 합니다.
